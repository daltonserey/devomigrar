Devo migrar para a nova grade?

Esta página foi criada pela coordenação de curso como uma forma de proporcionar orientação geral para os alunos na migração para o novo currículo do curso. A ideia orientar você com base no seu perfil de execução curricular, partindo de dois dados apenas: i) o número de períodos integralizados; e ii) o número de créditos integralizados. Você acha esses dois dados no seu histórico escolar. Digite os dados abaixo, para receber a orientação que melhor se adequa a seu caso. Leia a orientação com atenção antes de optar por migrar.
